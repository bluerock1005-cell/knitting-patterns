 # 项目更新日志
 
 记录本人编织图纸管理器的功能迭代与改进。
 
 ---
 
 ## 2026-07-03
 
 ### 卡片式 UI 重构
 
 将原有的表格视图（QTableWidget）全面替换为卡片瀑布流布局，提升浏览体验。
 
 **变更清单：**
 - 新增 `FlowLayout` 自定义布局，支持子控件自动换行排列
 - 新增 `PatternCard` 卡片组件，包含封面图、标题、分类标签、备注预览
 - 卡片悬浮时浮现编辑/删除按钮
 - 双击卡片直接进入编辑对话框
 - 「● 已上线」徽标标记已生成网页的图纸
 - 选中卡片高亮边框效果
 - 新增 5 种配色变量（棒针/钩针/成功/浅色背景等）
 
 **删除：**
 - 移除 `ComboBoxDelegate`、`LargeLineEditDelegate`（表格内联编辑不再需要）
 
 ### 封面图片手动上传
 
 添加/编辑对话框新增「📷 封面图片」区域，支持：
 - 🖼️ 浏览本地图片文件，自动复制到 `docs/images/`
 - ✕ 清除已选择的图片
 - 手动上传的图片优先级高于 Ravelry 自动下载
 
 ### PDF 管理 — 删除功能
 
 「PDF 文件」标签页新增「🗑️ 删除 PDF」按钮：
 - **未收录的 PDF**：仅删除文件，不影响 `patterns.csv`
 - **已收录的 PDF**：确认弹窗提示联动删除，同时清理 `patterns.csv` 中的图纸记录
 - 删除后自动刷新 PDF 列表和主界面卡片网格
 
 ### 生成网页防闪烁
 
 修复点击「生成网页」时左上角卡片闪现错位的问题。
 
 **优化方案：**
 - 生成完成后不再销毁并重建全部卡片
 - 改用 `_refresh_card_states()` 仅更新「已上线」徽标状态
 - `PatternCard.set_generated()` 动态创建/隐藏徽标控件
 - 全量重建卡片时包裹 `setUpdatesEnabled(False/True)` 防止渲染闪烁
